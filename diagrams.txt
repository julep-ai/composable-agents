<!-- AIDEV-NOTE: protocol-flow; sequence diagram showing MCP and A2A communication patterns -->
sequenceDiagram
    participant Agent1 as Agent 1
    participant MCP1 as MCP Server
    participant PGMQ as pgmq Queue
    participant MCP2 as MCP Client
    participant Agent2 as Agent 2
    participant A2A as A2A Protocol
    
    Note over Agent1,Agent2: MCP Communication
    Agent1->>MCP1: Request Tool
    MCP1->>PGMQ: Queue Message
    PGMQ->>MCP2: Deliver Message
    MCP2->>Agent2: Execute Tool
    Agent2-->>Agent1: Return Result
    
    Note over Agent1,Agent2: A2A Communication
    Agent1->>A2A: Discover Agents
    A2A-->>Agent1: AgentCard Registry
    Agent1->>A2A: Create Task
    A2A->>PGMQ: Queue Task
    PGMQ->>Agent2: Deliver Task
    Agent2->>A2A: Update Task Status
    A2A-->>Agent1: Task Complete
